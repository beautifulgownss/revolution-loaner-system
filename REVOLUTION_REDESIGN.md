## 🔧 Customization

### Change Primary Color
Edit `client/tailwind.config.js`:
```javascript
'revolution': {
  primary: '#2563eb', // Change this
  ...
}
```

### Modify Fonts
Edit `client/src/index.css`:
```css
@import url('https://fonts.googleapis.com/...');
```

### Adjust Animations
Edit animation timings in `tailwind.config.js`
